# Word テンプレート

このファイルは契約書の書式テンプレートです。

## カスタマイズ方法

1. このファイルを Microsoft Word または Google Docs で開く
2. 以下をカスタマイズ：
   - フォント（游明朝、MS 明朝など）
   - フォントサイズ（本文 10.5pt、見出し 12pt など）
   - 余白（上下左右 25mm など）
   - 行間（1.5倍など）
   - ヘッダー・フッター
3. 保存

## 適用されるスタイル

Pandoc が以下の Markdown 要素を Word スタイルに変換：

- `# 見出し1` → Heading 1
- `## 見出し2` → Heading 2
- `### 見出し3` → Heading 3
- 本文 → Normal
- リスト → List Paragraph

Word のスタイル機能を使って、各スタイルを定義してください。

経費精算規程

第1章 総則

第1条（目的）

本規程は、株式会社〇〇（以下「会社」という）の業務遂行に伴う経費の精算手続き、基準及び管理に関する事項を定めるものである。

第2条（適用範囲）

本規程は、会社に雇用される全ての従業員に適用する。ただし、取締役、監査役については、別に定める規定によるものとする。

第3条（経費の定義）

本規程において「経費」とは、会社の業務遂行に必要かつ合理的であると認められる支出をいう。

第2章 経費の区分

第4条（経費の種類）

経費は、次の通り区分する。

  1. 交通費: 業務上の移動に要する費用

    • 電車賃、バス代、タクシー代等

  2. 宿泊費: 業務上の出張に伴う宿泊費用

  3. 交際費: 取引先等との接待に要する費用

    • 食事代、手土産代等

  4. 会議費: 会議の開催に要する費用

    • 会場費、茶菓子代等

  5. 図書研究費: 業務上必要な図書、資料の購入費用

  6. 消耗品費: 業務上使用する消耗品の購入費用

  7. 通信運搬費: 郵便、電話、宅配料金等

  8. その他: 上記以外で、業務上必要と認められる費用

第3章 精算基準

第5条（交通費）

  1. 鉄道利用

    • 通常は普通乗車券とする
    • 距離が片道50kmを超える場合は、急行券・特急券を認める
    • 新幹線は、片道100kmを超える場合に認める（のぞみ・みずほを除く）

  2. 航空機利用

    • 距離が片道500kmを超える場合、又は緊急を要する場合に認める
    • クラスは、エコノミークラスとする

  3. タクシー利用

    • 終バス後、又は業務上やむを得ない場合に限る
    • 実費精算とする

  4. 自家用車利用

    • 業務上必要な場合、1kmあたり〇〇円を支給する
    • ガソリン代、駐車場代は実費精算とする

第6条（宿泊費）

  1. 宿泊費の上限

    • 東京・大阪・名古屋地区： 1泊12,000円まで
    • その他の地区： 1泊10,000円まで

  2. 宿泊費の精算

    • 実費精算とする
    • 上限を超える場合、事前に上司の承認を得るものとする

第7条（交際費）

  1. 交際費の基準

    • 接待相手1名につき、8,000円まで
    • 上限を超える場合、事前に部門長の承認を得るものとする

  2. 交際費の精算

    • 参加者名、接待相手名、目的を明記すること
    • 領収書を必ず添付すること

第8条（会議費）

  1. 会議費の基準

    • 参加者1名につき、1,000円まで
    • 上限を超える場合、事前に上司の承認を得るものとする

  2. 会議費の精算

    • 会議名、参加者名、日時を明記すること

第9条（図書研究費）

  1. 図書研究費の基準

    • 1冊につき5,000円まで
    • 上限を超える場合、事前に上司の承認を得るものとする

  2. 図書研究費の精算

    • 購入目的を明記すること
    • 購入図書は、会社の資産とする

第10条（消耗品費）

  1. 消耗品費の基準

    • 1回につき10,000円まで
    • 上限を超える場合、事前に上司の承認を得るものとする

  2. 消耗品費の精算

    • 購入品目、数量、用途を明記すること

第4章 精算手続き

第11条（精算書の提出）

  1. 経費精算書は、毎月末日までに提出するものとする
  2. 遡及精算は、原則として翌月末までとする

第12条（精算書の記載事項）

精算書には、次の事項を記載するものとする。

  1. 支出年月日
  2. 支出先
  3. 経費の種類
  4. 金額
  5. 支出目的・事由
  6. 勘定科目

第13条（添付書類）

精算書には、次の書類を添付するものとする。

  1. 領収書（必須）
  2. その他、支出を証明する書類（必要に応じて）

第14条（承認手続き）

  1. 精算書は、所属長の承認を得るものとする
  2. 1件10万円を超える精算は、部門長の承認を得るものとする
  3. 1件50万円を超える精算は、取締役の承認を得るものとする

第5章 支払

第15条（支払時期）

経費は、精算書提出の翌月末日に支払うものとする。

第16条（支払方法）

経費は、従業員の指定する金融機関の口座へ振り込むものとする。

第6章 管理と監督

第17条（経費の適正管理）

従業員は、経費を適正に管理し、不正、乱用を防止しなければならない。

第18条（監査）

会社は、経費精算について監査を行うことができる。従業員は、監査に協力しなければならない。

第19条（不正精算の処置）

不正精算が発覚した場合、会社は次の措置を講じることができる。

  1. 不正金額の返還請求
  2. 懲戒処分（譴責、減給、出勤停止、解雇等）
  3. 刑事告訴

第7章 雑則

第20条（規程の変更）

本規程の変更は、取締役会の決議により行う。

第21条（細則）

本規程の実施に関し必要な事項は、別に定める。

第22条（未定事項）

本規程に定めのない事項は、法令の規定及び会社の内規に従う。

附則

  1. 本規程は、2025年4月1日より施行する。
  2. 従前の経費精算規程は、廃止する。

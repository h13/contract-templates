出退勤管理規程

第1章 総則

第1条（目的）

本規程は、株式会社〇〇（以下「会社」という）における従業員の出退勤管理に関し、その方法および手続き等を定めることを目的とする。

第2条（適用範囲）

本規程は、会社に雇用される全ての従業員に適用する。

第3条（定義）

本規程における用語の定義は、次のとおりとする。

  1. 出勤: 所定の始業時刻までに業務を開始すること
  2. 退勤: 所定の終業時刻以降に業務を終了すること
  3. 所定労働時間: 就業規則で定める労働時間
  4. 時間外労働: 所定労働時間を超える労働
  5. 休日労働: 所定休日における労働

第2章 出退勤の記録

第4条（打刻義務）

従業員は、出勤時および退勤時に、所定の方法により打刻しなければならない。

  1. 出勤時： 始業時刻前に打刻する
  2. 退勤時： 終業時刻後に打刻する
  3. 休憩開始時および終了時： 所定の方法により記録する

第5条（打刻方法）

出退勤の打刻は、次のいずれかの方法により行う。

  1. タイムカードによる打刻
  2. ICカードによる打刻
  3. 指紋認証等の生体認証による打刻
  4. 会社が指定するその他の方法

第6条（打刻の代行禁止**

従業員は、他の従業員に代わって打刻すること、または他の従業員に打刻を代行させることを禁止する。

第3章 遅刻・早退・欠勤

第7条（遅刻）

始業時刻までに出勤しなかった場合、遅刻とみなす。遅刻した従業員は、直ちに上司にその旨を報告しなければならない。

第8条（早退）

終業時刻前に退勤する場合、早退とみなす。早退する従業員は、事前に上司の承認を得なければならない。

第9条（欠勤）

勤務すべき日に勤務しない場合、欠勤とみなす。欠勤する従業員は、次のとおり手続きを行わなければならない。

  1. 病気または負傷による欠勤： 始業時刻までに上司に連絡する
  2. 事由による欠勤： 原則として前日までに上司の承認を得る
  3. その他の事由： 直ちに上司に連絡し、事後に欠勤届を提出する

第10条（遅刻・早退・欠勤の給与**

遅刻・早退・欠勤による減給は、就業規則および給与規程の定めるところによる。

第4章 休憩・休息

第11条（休憩）

会社は、労働基準法の定めるところにより、次の休憩を与える。

  1. 労働時間が6時間を超える場合： 45分以上
  2. 労働時間が8時間を超える場合： 1時間以上

第12条（休憩の取得方法）

休憩は、一括して取得することを原則とする。ただし、業務の都合により分割して取得する場合は、上司の承認を得るものとする。

第13条（休息）

業務の性質上、連続して長時間労働する場合、適宜休息をとるものとする。

第5章 時間外労働・休日労働

第14条（時間外労働の命令）

業務の都合により必要がある場合、会社は従業員に時間外労働を命ずることができる。この場合、従業員は正当な理由なく拒否してはならない。

第15条（時間外労働の手続き）

時間外労働を行う場合、従業員は事前に時間外労働届を提出し、上司の承認を得なければならない。

第16条（休日労働の手続き）

休日に労働する場合、従業員は事前に休日労働届を提出し、上司の承認を得なければならない。

第17条（時間外労働・休日労働の限度）

時間外労働および休日労働は、労働基準法および関連法規の定める限度を超えないものとする。

第6章 勤務状況の確認

第18条（勤務表の作成）

人事部は、毎月従業員の勤務状況を記載した勤務表を作成し、これを保管する。

第19条（勤務状況の確認）

従業員は、毎月の勤務状況を確認し、内容に誤りがある場合は、速やかに人事部に申し出るものとする。

第20条（勤怠記録の保存**

勤怠記録は、労働基準法の定めるところにより、3年間保存する。

第7章 出張

第21条（出張の手続き）

出張する場合、従業員は事前に出張届を提出し、上司の承認を得なければならない。

第22条（出張中の出退勤**

出張中の出退勤については、出張先の状況に応じて適切に記録するものとする。

第8章 雑則

第23条（細則）

本規程の実施に関し必要な事項は、別に定める。

第24条（規程の改廃）

本規程の改廃は、取締役会の決議による。

第25条（施行日）

本規程は、2025年4月1日から施行する。
